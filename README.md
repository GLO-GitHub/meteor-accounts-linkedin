meteor-accounts-linkedin
============================

A meteor package for LinkedIn's login service.

Package Dependencies
----------------------

* accounts-base
* accounts-oauth
* pauli:linkedin

Install
-----------
```
meteor add pauli:accounts-linkedin
```
